# SupplyStack 📦
**Smart Inventory Management System**

SupplyStack is a decoupled web application designed for restaurant inventory management. It features a robust Python/Django REST API backend and a lightweight, responsive HTML/CSS/JavaScript frontend dashboard.

## 🚀 Tech Stack
* **Backend:** Python, Django, Django REST Framework, SQLite
* **Frontend:** HTML5, CSS3, Vanilla JavaScript
* **Data Visualization:** Chart.js

## 📂 Project Structure
This project uses a decoupled architecture, separating the client interface from the database layer.
* `/SupplyStackAPI/` - Contains the Django backend, models, serializers, and API views.
* `/demoSupplyStackFrontend/` - Contains the HTML dashboard and JavaScript fetch logic.
